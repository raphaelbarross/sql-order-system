# Sistema de pedidos SQL

Projeto simples de banco de dados em SQL que simula um sistema básico de pedidos de uma loja.

## Sobre o projeto

Este projeto foi criado para praticar conceitos fundamentais de banco de dados, como:

* criação de tabelas
* inserção de dados
* consultas em SQL

O banco de dados simula uma pequena loja com clientes, produtos e pedidos.

## Estrutura do banco de dados

O banco possui três tabelas principais:

* clientes — armazena informações dos clientes
* produtos — armazena os produtos da loja
* pedidos — registra os pedidos feitos pelos clientes

As tabelas se relacionam através dos IDs dos registros.

## Tecnologias utilizadas

* SQL
* PostgreSQL



## Como executar o projeto

1. Abra o PostgreSQL
2. Crie um banco de dados
3. Execute o arquivo loja.sql

O script irá:

* criar as tabelas
* inserir dados de exemplo
* permitir consultas no banco
